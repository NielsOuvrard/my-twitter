# Website Free

Complet website hosted on a free server (for free).

## [Website](http://ouvrard.niels.free.fr/)

## Frontend

### Architecture

```sh
./frontend
├── babel.config.js
├── package-lock.json
├── package.json
├── tsconfig.json
├── vue.config.js
│
├── public
│   ├── favicon.ico
│   ├── img
│   │   └── icons
│   │       └── ...
│   ├── index.html
│   └── robots.txt
│
└── src
    ├── App.vue
    ├── assets
    │   └── logo.png
    ├── components
    │   └── ...
    ├── main.ts
    ├── note.txt
    ├── registerServiceWorker.ts
    ├── router
    │   └── index.ts
    ├── shims-vue.d.ts
    ├── store
    │   └── index.ts
    └── views
        └── ...
```

## Backend

### Architecture

./backend

│

│   └── RelationshipController.php

├── routes

│
└── utils
    ├── db.php
